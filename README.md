# angular2withspringboot
A sample web application with angular2 front end and spring boot backend.
************How to run application******************
1. As a composite application:
   angular2springboot>mvn clean install
   angular2springboot>cd backend
   angular2springboot\backend>mvn spring-boot:run
   
2. As separate modules:
   angular2springboot\frontend\src\main\frontend>npm start
   angular2springboot\backend>mvn spring-boot:run
